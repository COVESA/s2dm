The _Simplified Semantic Data Modeling (`S2DM`)_ approach is rather simple and involves the following personas:
* **Domain Expert** A domain expert modeling the domain in a machine-readable human-friendly syntax.
Then, we can use the tools on that model for common governance tasks.
Finally, we export the model into a particular format that is needed in the downstream system.
